# Bando Omni-AGI Pipeline

Drop-in AGI architecture for chaining multimodal agents (Qwen-VLo, TOWER+, Magenta, EmbodiedGen) under Victor's orchestration.

- Plug in your keys/API endpoints to config.yaml
- Flesh out agent stubs as you wire up real models/APIs
- `pipeline.py` = main entry point

All code is modular, future-proof, and savage.
